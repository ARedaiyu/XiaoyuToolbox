#### 1.0.3-Release1 首个正式版

《小鱼工具箱》是一个基于 WPF 框架（.NET）和C#语言开发的综合工具箱桌面应用程序。

工具箱内包含多个工具：

* 编码工具，包含《文本转Unicode》
* 计算器，包含《进制转换1》《日期计算器》和《平均数计算器》
* 随机工具，包含《随机密码生成器》
* 设备检测，包含《鼠标检测》

项目使用了 NuGet 包 CommunityToolkit.Mvvm，基于MVVM模式开发。

该程序需要在Windows 10 / 11操作系统下运行。需要安装 .NET 8.0 运行时。

.NET 下载地址（简体中文页面）： https://dotnet.microsoft.com/zh-cn/download/dotnet

技术栈：

* .NET 8.0
* WPF (Windows Presentation Foundation)
* MVVM
* NuGet 包 CommunityToolkit.Mvvm
* C# 语言

源码地址： https://github.com/ARedaiyu/XiaoyuToolbox

该项目为免费开源项目，仅供学习与个人使用。
